# Değişiklik günlüğü

## 0.1.0

- Alt para birimi tabanlı para ve açık yuvarlama tipleri eklendi.
- Avalanche ve snowball borç simülasyonu eklendi.
- Kapsama güven kapılı Laspeyres endeksi eklendi.
- SHA-256 ve Ed25519 doğrulamalı LightGBM çıkarımı eklendi.
- İmzalı sürüm 2 manifest ve cihaz içi olasılık kalibrasyonu eklendi.
- Cholesky çözümlü, idempotent ödüllü LinTS politikası eklendi.
- Flutter, ağ ve veritabanı bağımlılığı taşımayan dışa açık API tamamlandı.
